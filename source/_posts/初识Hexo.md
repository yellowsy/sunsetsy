#### 1.全局安装hexo命令
```shell
npm install -g hexo-cli
```
#### 2.创建新文章
```
hexo new "文章名称"
```
#### 3.本地启动
在博客根目录下运行命令：
```shell
hexo s
```
#### 4.清理缓存
```shell
hexo clean
```
#### 5.生成静态页面
```shell
hexo g
```
#### 6.上传到仓库
```shell
hexo d
```
